== Introduction

This page serves as the central hub for all Apache Shiro documentation. Whether you are just getting started or looking for detailed reference material, you will find links to the appropriate resources here.

=== New to Shiro?

If you are new to Apache Shiro, we recommend reading these resources in order:

* https://www.infoq.com/articles/apache-shiro[Application Security with Apache Shiro] - full intro article on InfoQ.com
* link:10-minute-tutorial.html[10 Minute Tutorial] - quick hands-on introduction
* link:webapp-tutorial.html[Beginner's Webapp Tutorial] - step-by-step guide to enable Shiro in a web application
* https://start.flowlogix.com[Flowlogix Starter] - generate Jakarta EE projects with Shiro support

Once you understand the basics, explore the link:guides.html[Guides] page for a complete learning path.
